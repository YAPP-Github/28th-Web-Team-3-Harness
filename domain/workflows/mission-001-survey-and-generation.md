# 미션 설문 및 추천 미션 생성 플로우

- ID: FLOW-MISSION-001
- 참여자: 사용자, 클라이언트, 미션 서비스
- 시작 조건: 사용자가 미션 설문을 작성하거나 추천 미션 생성을 요청함
- 종료 상태: 유효한 설문이 저장되었거나, 추천 미션이 확정되었거나, 요청이 거절됨

## 정상 흐름

1. 사용자가 하나 이상의 설문 카테고리를 선택하고 질문 정의에 맞춰 응답한다.
2. 서비스가 응답의 선택·조건·값 범위를 검증한 뒤 사용자의 기존 설문 전체를 새 응답으로 교체한다.
3. 사용자가 추천 미션 생성을 요청하면 서비스는 생성 선행 조건과 설문 존재 여부를 확인한다.
4. 진행 중인 생성 작업이 없으면 생성 작업을 만들고 미션 초안을 준비한다. 있으면 기존 작업을 반환한다.
5. 작업이 성공하면 사용자는 유효기간 안에 미션 초안을 확인한다.
6. 사용자가 서로 다른 초안 1~4개를 선택해 확정하면, 초안은 추천 미션이 된다.

## 예외 흐름

- 설문 응답이 질문의 조건을 만족하지 않으면 저장하지 않는다.
- 설문이 없거나 생성 선행 조건을 만족하지 않으면 미션 생성 요청을 거절한다.
- 작업이 대기·실행 중·실패 상태이거나 초안이 만료되었으면 초안 조회와 확정을 거절한다.
- 이미 확정한 작업에 같은 초안 집합을 다시 제출하면 기존 결과를 반환하고, 다른 집합이면 거절한다.
