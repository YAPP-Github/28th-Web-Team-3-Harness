# 미션 수행 및 주간 종료 플로우

- ID: FLOW-MISSION-002
- 참여자: 사용자, 클라이언트, 미션 서비스, 주간 종료 작업
- 시작 조건: 추천 또는 수동 미션이 활성 상태로 존재함
- 종료 상태: 미션이 완료 또는 미완료 상태가 됨

## 정상 흐름

1. 사용자는 자신의 추천 미션과 수동 미션을 함께 조회한다.
2. 사용자가 수동 미션을 만들면 해당 미션은 활성 상태로 이번 주 종료 시각까지 유지된다.
3. 사용자가 활성 미션을 완료하면 서비스는 미션을 완료 상태로 바꾸고 미션 결과를 기록한다.
4. 매주 월요일 00:00(Asia/Seoul)에 종료 시각을 지난 활성 미션은 미완료 상태로 바뀌고 미션 결과가 기록된다.

## 예외 흐름

- 다른 사용자의 미션, 존재하지 않는 미션, 이미 미완료인 미션은 완료할 수 없다.
- 이미 완료된 미션의 완료 요청은 상태를 중복 변경하지 않는다.
- 완료 요청과 주간 종료가 경합하면 두 작업 중 먼저 확정된 종료 상태를 유지한다.
